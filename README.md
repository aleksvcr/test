# Casa Seijaku — VS Code Export

This is a plain HTML and CSS landing page. It does not require React, Node.js, or package installation.

## Open it in VS Code

1. Open VS Code.
2. Choose **File → Open Folder**.
3. Select the `casa-seijaku-vscode-export` folder.
4. Open `index.html`.

For live preview, install the **Live Server** extension, right-click `index.html`, and select **Open with Live Server**.

## Project files

- `index.html` — all page text and sections
- `styles.css` — layout, colors, typography and responsive behavior
- `assets/` — local Casa Seijaku images used by the page

To change the palette, edit the color variables at the top of `styles.css` under `:root`.
